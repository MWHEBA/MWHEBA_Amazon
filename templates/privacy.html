سياسة الخصوصية

آخر تحديث: {% now "j F Y" %}

تصف سياسة الخصوصية هذه كيفية جمع واستخدام وحماية المعلومات الشخصية التي تقدمها عند استخدام نظام مزامنة FBM ("النظام"). نحن ملتزمون بضمان خصوصية بياناتك.

1. المعلومات التي نجمعها

قد نجمع المعلومات التالية:

  • معلومات التعريف الشخصية (مثل الاسم والبريد الإلكتروني ورقم الهاتف)
  • معلومات حساب البائع على أمازون (مثل بيانات اعتماد API ومعرفات المنتجات)
  • بيانات المخزون والمنتجات
  • معلومات الاستخدام والتفاعل مع النظام
  • معلومات الجهاز وعنوان IP

2. كيف نستخدم المعلومات

نستخدم المعلومات التي نجمعها للأغراض التالية:

  • توفير وإدارة وتحسين خدماتنا
  • مزامنة المخزون مع حساب البائع الخاص بك على أمازون
  • التواصل معك بشأن حسابك والتحديثات والتغييرات في خدماتنا
  • تقديم الدعم الفني
  • تحليل استخدام النظام وتحسين تجربة المستخدم

3. أمان البيانات

نحن ملتزمون بضمان أمان معلوماتك. لمنع الوصول غير المصرح به أو الكشف عن معلوماتك، قمنا بتنفيذ إجراءات مادية وإلكترونية وإدارية مناسبة لحماية المعلومات التي نجمعها عبر الإنترنت.

4. بيانات اعتماد Amazon API

نقوم بتخزين بيانات اعتماد Amazon API الخاصة بك (مثل مفاتيح الوصول وعلامات التحديث) بشكل آمن ومشفر. نستخدم هذه البيانات فقط للاتصال بـ Amazon SP-API نيابة عنك لمزامنة معلومات المخزون.

5. مشاركة البيانات

لا نبيع أو نتاجر أو ننقل معلوماتك الشخصية إلى أطراف خارجية إلا في الحالات التالية:

  • عندما نستخدم خدمات أطراف ثالثة موثوقة لمساعدتنا في تشغيل النظام (مثل استضافة قواعد البيانات)
  • عندما نعتقد أن الإفصاح مناسب للامتثال للقانون أو إنفاذ سياسات موقعنا
  • لحماية حقوقنا أو ممتلكاتنا أو سلامتنا أو سلامة الآخرين

6. ملفات تعريف الارتباط

قد نستخدم ملفات تعريف الارتباط لتحسين تجربة المستخدم وتحليل استخدام النظام. يمكنك اختيار رفض ملفات تعريف الارتباط من خلال إعدادات المتصفح الخاص بك.

7. حقوقك

لديك الحق في:

  • الوصول إلى المعلومات الشخصية التي نحتفظ بها عنك
  • تصحيح أي معلومات غير دقيقة
  • طلب حذف معلوماتك الشخصية
  • الاعتراض على معالجة معلوماتك الشخصية
  • طلب تقييد معالجة معلوماتك الشخصية
  • طلب نقل معلوماتك الشخصية

8. الاحتفاظ بالبيانات

سنحتفظ بمعلوماتك الشخصية فقط طالما كان ذلك ضروريًا للأغراض المنصوص عليها في سياسة الخصوصية هذه. عند إلغاء حسابك، سنحذف معلوماتك الشخصية في غضون 30 يومًا، باستثناء المعلومات التي يتعين علينا الاحتفاظ بها لأسباب قانونية أو تنظيمية.

9. التغييرات في سياسة الخصوصية

قد نقوم بتحديث سياسة الخصوصية هذه من وقت لآخر. سنخطرك بأي تغييرات عن طريق نشر السياسة الجديدة على هذه الصفحة. يرجى مراجعة هذه الصفحة بشكل دوري للتأكد من أنك راضٍ عن أي تغييرات.
